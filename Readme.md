This website is built with HTML, CSS, and JavaScript, IN addition to the following libraries:
- [Bootstrap](https://getbootstrap.com/)
- [FontAwesome](https://fontawesome.com/)
- [jQuery](https://jquery.com/)
The entire website is hosted on [Render](https://render.com/).
The database is hosted on [Railway](https://railway.app/).

The website is available at [https://interior-design-1-tie5.onrender.com](https://interior-design-1-tie5.onrender.com).

Note: the runtime environment for the website is Node.js 18.x.
